# plasma-blossom-patches

When creating the iconpack for [BlossomUI](https://codeberg.org/BlossomOS/ui) we noticed that KDE Plasma only supports battery icons in 10 percent steps but we wanted at least 5% steps.
We also wanted the wider icon to fit better into the bar instead of having a 1:1 aspect ratio.
Then we noticed the systemtray needed more visual changes.
And the plasma workspace widget settings screen needed a reset to defaults button.
